Usage:
clip ( hither | yon ) distance [ fromCenter true | false ] [ frames ]

Usage:
~clip ( hither | yon )

Usage:
clip ( on | off )

Clip can be used to move the hither (front) or yon (back) global clipping plane a specified distance from its current position (default), or to a target location that is distance away from the center of rotation (with fromCenter true). The distance is in physical units of length, usually Å; positive values are toward the user, negative values away from the user. The back plane is automatically pushed slightly behind the front plane (0.01 distance units) if the specified movement would have reversed their positions.

The frames option specifies a number of image update frames (default 1). For frames > 1:

  • a movement of distance from the current clipping plane position (fromCenter false, default) is repeated at each frame
  • a movement to the target location of distance from the center of rotation (fromCenter true) is performed gradually over the total number of frames
See the video mini-example. See also: cofr, section, thickness, mclip, sop cap, the Side View, movie-related commands

~Clip halts an ongoing clip plane motion. Commands continue to be processed while the requested motion is in progress. To pause command processing until the motion is finished, use the wait command.

A second use of clip is to turn global clipping on and off. Turning it on does not restore previous clipping plane positions.
